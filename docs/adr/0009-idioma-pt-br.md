# ADR-0009: Idioma — pt-BR

- **Status:** aceito (2026-06-21)

## Contexto

Audiência = recrutador/par técnico. A copy seria prototipada já. Idioma afeta tom, comprimento e layout.

## Decisão

panlabs fala **pt-BR** no V1.

## Consequências

- Autêntico, esforço baixo, casa com o ecossistema do autor.
- Limita alcance internacional — **EN/bilíngue** ficam registrados pro V2.

## Alternativas rejeitadas (por ora)

- **Inglês** — alcance internacional/remoto maior, mas custa naturalidade.
- **Bilíngue (toggle)** — dobro de copy + i18n a zero usuário.
