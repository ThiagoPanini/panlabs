# ADR-0007: Audiência e posicionamento — laboratório vivo

- **Status:** aceito (2026-06-21)

## Contexto

O prompt original era ambíguo: *"vitrine para um usuário que queira conhecer minhas soluções"* mas *"criadas majoritariamente para meus próprios propósitos"* e *"sem usuários ativos, sem expectativa de crescer"*. Audiência diferente = design radicalmente diferente.

## Decisão

- **Audiência primária:** pares técnicos e recrutadores.
- **Posicionamento:** **laboratório vivo** — experimentos AI-built que viraram SaaS reais; técnico com substância + toque lúdico; premium **sem fingir escala**.
- **Idioma:** pt-BR (ver [ADR-0009](0009-idioma-pt-br.md)).

## Consequências

- O design serve **credibilidade/reputação**, não conversão. Sem CTAs de funil, sem simular usuários/escala.
- O capricho visual é justificado por ser uma **vitrine de reputação** — coerente com o card escolhido em [ADR-0005](0005-indicadores.md).

## Alternativas rejeitadas

- **Usuários-fim / funil** — não há fluxo a converter.
- **Você-mesmo / cockpit pessoal** — "Premium" seria overkill para plateia de um.
- **Studio de produtos** — finge escala/empresa que não existe; briga com o nome "labs".
- **Só engenharia / craft** — árido, sério demais, ignora o convite lúdico de "labs".
- **Comunidade / build-in-public** — custo recorrente de conteúdo/narrativa.
